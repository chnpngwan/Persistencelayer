# FluentMybatis: 基于mybatis但青出于蓝
- **No XML, No Mapper, No If else, No String魔法值编码**
- **只需Entity就实现强大的FluentAPI: 支持分页, 嵌套查询, AND OR组合, 聚合函数...**
